``--except-a2l`` |image_advanced_argument|
""""""""""""""""""""""""""""""""""""""""""


Enhance the backtrace when displaying exception by changing program addresses into file names and lines (may take some seconds).

``--except-no-bt`` |image_advanced_argument|
""""""""""""""""""""""""""""""""""""""""""""
